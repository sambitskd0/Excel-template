
# Eshikshakosh



#### Excel template generation in Angular

- Install excel js and file saver package

```bash
 npm i exceljs
 npm i file-saver
```
- Set below code for excel js compiler option in tsconfig.json file

```bash
    "paths": {
      "exceljs": [
        "node_modules/exceljs/dist/exceljs.min"
      ]
    }

```
